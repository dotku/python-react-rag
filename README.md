# Pthon React RAG

## Preperation

1. setup your pincone account and fill up the evn

## Get Started

start backend service

```sh
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

start frontend client

```sh
cd ../frontend
npm install
npm start
```

happy to play
